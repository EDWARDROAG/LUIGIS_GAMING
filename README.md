# 🎮 Gaming Store

## Tienda de Videojuegos, Consolas y Reparación

### 🚀 Tecnologías
- **Frontend:** React + Vite + TailwindCSS
- **Backend:** Node.js + Express
- **Monorepo:** npm workspaces

### 🛠️ Instalación
```bash
npm install
cd apps/web && npm run dev
cd apps/api && npm run dev
```
